Smash Soda must be built from source due to licensing issues. 

## Installer

The easiest way to do this is with the installer batch script at:

[https://github.com/soda-arcade/smash-soda/releases/latest/installer.bat](https://github.com/soda-arcade/smash-soda/releases/latest/installer.bat)

Download this, then right click it and select "Run as administrator". The script will take you through a guided setup.

## Manual build (if installer fails)

If the installer does not work on your machine, follow the manual guide:

[Manual Build (Windows)](manual-build.md)
